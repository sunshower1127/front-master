# Svelte 마스터 로드맵

React 경험자 → Svelte 마스터. **E-commerce(쇼핑몰)** 프로젝트를 처음부터 끝까지 만들면서 배움.

## 프로젝트: 쇼핑몰

하나의 프로젝트를 마일스톤마다 키워감. 각 단계에서 새로운 기능을 추가하면서 Svelte 개념을 자연스럽게 익힘.

| 02  | [리액티비티](m02-reactivity.md)      | 장바구니, 필터/정렬        | 전역 상태, 깊은 리액티비티, 컴파일러 | ★★     |

## 왜 이 순서인가

1. **M01~M02**: Svelte 핵심을 쇼핑몰 기본 기능(상품 카드, 장바구니)으로 체감
2. **M03**: 정적 데이터 → API 데이터로 전환. 실제 앱처럼 동작하게
3. **M04**: 기능이 늘어나기 전에 구조부터 잡음 (기존 m08에서 앞으로 이동)
4. **M05~M06**: 검색엔진 노출 + 에셋 최적화. 쇼핑몰의 필수 요소
5. **M07~M08**: 규모가 커진 후 성능 + 안정성 확보
6. **M09**: 완성된 쇼핑몰을 실제 배포

## 진행 방식

- 각 마일스톤에서 쇼핑몰에 기능을 하나씩 추가
- 이전 마일스톤의 코드 위에 계속 쌓아감
- 과제를 완료하면 검증 방법으로 확인
- 막히면 해당 마일스톤 문서에 메모 남기고 다음으로 넘어가도 됨
